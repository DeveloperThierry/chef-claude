#### Prerequisites

- npm or yarn

#### Installation

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/your-username/chef-claude.git](https://github.com/your-username/chef-claude.git)
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or

    ```

3.  **Run the application in development mode:**
    ```bash
    npm start
    # or
    yarn start
    ```
